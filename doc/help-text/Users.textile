h1. Users

The user list is split into three categories:

* ACTIVE: Users who can use the system
* BLOCKED: Users who are temporarily blocked from using the system
* DELETED: Users who have been deleted.

Once a user has been created, their account will remain in the system to ensure that historical data is maintained. 

h2. New user

* Click _New user_ to create a new user.
